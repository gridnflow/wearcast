📸 캡처 방법 (How to capture)

  1. Chrome에서 이 페이지 열기 → 상단 언어 버튼으로 언어 선택
  2. 캡처할 스크린샷 위에서 우클릭 → 검사(Inspect)
  3. .phone 요소 선택 → 우클릭 → Capture node screenshot
  4. 또는 Chrome DevTools 모바일 에뮬레이터에서 390×844로 설정 후 전체 페이지 캡처
  5. 각 언어별로 1~6번 반복 → Play Console 언어별 탭에 업로드

💡 Play Store 권장 사이즈: 1080×1920px (9:16) · 최소 2장, 최대 8장 · JPG 또는 PNG
