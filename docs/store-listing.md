## 📋 Play Console 입력 순서 가이드

1. Play Console → 앱 선택 → **Store presence** → **Main store listing**
2. 상단 언어 탭에서 각 언어 추가
3. **App name**: `android/app/src/main/res/values-{lang}/strings.xml` 값과 동일하게 입력
4. **Short description**: 위 "짧은 설명" 복붙
5. **Full description**: 위 "전체 설명" 복붙
6. 저장 후 **Review and publish**
