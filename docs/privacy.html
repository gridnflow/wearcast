개인정보 처리방침

Wearcast · 시행일: 2026년 4월 27일

Gridnflow('이하 "회사")는 날씨 기반 옷차림 추천 앱 Wearcast의 운영과 관련하여 「개인정보 보호법」 및 관련 법령을 준수하며, 이용자의 개인정보를 아래와 같이 처리합니다.

1. 수집하는 개인정보 항목

항목 수집 방법 목적
기기 위치 정보 (위도·경도) 기기 GPS (이용자 허용 시) 현재 날씨 및 옷차림 추천
앱 사용 이벤트 (화면 조회, 탭 등) Firebase Analytics 자동 수집 서비스 개선 및 통계 분석
기기 정보 (OS 버전, 앱 버전) Firebase Analytics 자동 수집 오류 분석 및 호환성 확인
설정 값 (온도 단위, 체감 옵션) 기기 로컬 저장 사용자 설정 유지

위치 정보는 날씨 API 조회 후 서버에 저장하지 않으며, 기기 내에서만 처리됩니다.

2. 개인정보의 처리 목적

  • 현재 위치 기반 실시간 날씨 정보 제공
  • 날씨·체감온도에 따른 옷차림 추천
  • 앱 서비스 품질 개선 및 신기능 개발

3. 개인정보의 보유 및 이용 기간

위치 정보는 요청 처리 즉시 파기되며 서버에 저장되지 않습니다.
Firebase Analytics 데이터는 Firebase 정책에 따라 최대 14개월 보관 후 자동 삭제됩니다.

4. 제3자 서비스 이용

서비스제공 정보목적
OpenWeatherMap API 위도·경도 날씨 데이터 조회
Google Firebase Analytics 앱 이벤트, 기기 정보 사용 통계 분석

각 서비스의 개인정보 처리방침:
· OpenWeatherMap: openweathermap.org/privacy-policy
· Google: policies.google.com/privacy

5. 이용자의 권리

  • 기기 설정에서 위치 권한을 언제든지 철회할 수 있습니다 (단, 서비스 이용 불가).
  • 앱을 삭제하면 기기에 저장된 모든 설정 데이터가 삭제됩니다.
  • Firebase Analytics 수집을 원하지 않으시면 아래 이메일로 문의해 주세요.

6. 어린이 개인정보

본 앱은 만 14세 미만 어린이를 대상으로 하지 않으며, 의도적으로 어린이의 개인정보를 수집하지 않습니다.
